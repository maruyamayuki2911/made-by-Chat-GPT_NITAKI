# ✅ 要件定義書：煮焚（NITAKI） サンプルWebサイト

---

## 1. サイトの目的

* ラーメン店「煮焚」の**店舗紹介**を目的とした静的Webサイトの作成。

---

## 2. 対象ユーザー

* ラーメン好き
* 近隣住民
* 観光客

---

## 3. 対応デバイス

* **PCのみ対応**

  * スマートフォンやタブレットへの対応（レスポンシブデザイン）は行わない。

---

## 4. デザイン方針

* **テイスト：** 落ち着いた／モダン／和風
* **カラー：** 添付画像に準拠（例：白背景＋黒文字、アクセントに茶・赤系など）
* **フォント：**

  * `"Sawarabi Mincho", serif`
  * フォントサイズは一般的なWebサイトに準拠

---

## 5. レイアウト仕様

* **固定レイアウト（ワンサイズ）**

  * 幅はPC画面を基準にデザイン（例：1200px前後）
* ページ内はスクロールで遷移する**縦長1ページ構成**
* ナビゲーションリンクにより該当セクションへ**スムーススクロール**可能

---

## 6. ページ構成・コンテンツ

| セクション名  | 内容                                      |
| ------- | --------------------------------------- |
| ホーム     | メインビジュアル（ラーメンの写真）、ロゴ、ナビゲーションバー          |
| 煮焚のこだわり | 出汁・素材・製法・麺に関するこだわりを4枚の画像と短文で構成          |
| お品書き    | ラーメンメニュー6種類（画像＋説明＋価格）＋トッピング・ご飯物などの一覧    |
| 店舗情報    | 住所／電話番号／営業時間／最寄駅／地図（Google Map埋め込み）     |
| フッター    | コピーライト、SNSアイコン（リンクあり）、ページ上部へのスクロールボタンなど |

---

## 7. コンテンツ素材

* **画像：** サンプル画像を仮使用
* **テキスト：** 提供された仮文言を使用（変更不要）

---

## 8. 技術要件

* 使用技術：

  * HTML5
  * CSS3（プレーンCSS。フレームワーク不使用）
* JavaScript：使用予定なし（必要に応じて最小限のスムーススクロールなど）

---

## 9. 公開環境

* **GitHub Pages** にて静的ホスティング

---

## 10. SEO / SNS連携

* **SNSリンク埋め込みのみ**

  * Instagram／X（旧Twitter）アイコン＋外部リンク
* SEO対策は今回不要

---

## 11. 保守・運用

* 本サイトは**サンプル用**のため、今後の更新等は考慮不要
